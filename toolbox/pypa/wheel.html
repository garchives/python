Wheel

Wheel

根据 application 包含的代码类型以及其所支持的 python 版本， wheel 格式可细分为三种

Universal wheel: 纯 python 代码，并且支持 python 2 和 3 Pure python wheel: 纯 python 代码，不同时支持 python2 和 3 Platform wheel: 非纯 python 代码
